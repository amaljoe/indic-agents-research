# Memory in the Age of AI Agents: A Survey

**Authors:** Yuyang Hu et al. (47 authors)

**Venue:** arXiv Survey (2025)

**ArXiv:** [2512.13564](https://arxiv.org/abs/2512.13564)

---

## Abstract Summary

This comprehensive survey provides a structured taxonomy of agent memory systems along three orthogonal dimensions: **forms** (token-level, parametric, latent memory), **functions** (factual, experiential, working memory), and **dynamics** (formation, evolution, retrieval). It compiles existing benchmarks and frameworks, critically examines current agent memory architectures, and articulates emerging research frontiers including memory automation, multimodal memory, and the interplay between memory and reasoning. The survey spans 200+ papers and is the most thorough treatment of LLM agent memory to date.

---

## Key Contributions

- **Unified taxonomy**: Three-dimensional classification of memory forms, functions, and dynamics
- **Token-level memory**: Analysis of in-context, KV-cache, and streaming memory approaches
- **Parametric memory**: Coverage of LoRA, model editing, and continual learning as memory mechanisms
- **Latent memory**: Embedding-based external memory stores (vector DBs, episodic memory)
- **Benchmark compilation**: Summary of evaluation frameworks for agent memory
- **Open problems**: Identifies memory automation, multimodal memory, and long-horizon consistency as frontiers

## Relevance to indic-agents

Provides the **conceptual foundation** for the project's three-layer memory architecture (textual, parameterized internal, parameterized external). The taxonomy directly maps to the project's memory components and informs the design of multilingual memory management strategies.
