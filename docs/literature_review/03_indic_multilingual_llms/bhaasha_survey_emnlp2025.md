# Bhaasha, Bhāṣā, Zaban: A Survey for Low-Resourced Languages in South Asia

**Authors:** Sampoorna Poria, Xiaolei Huang

**Venue:** EMNLP 2025 (Findings)

**ArXiv:** [2509.11570](https://arxiv.org/abs/2509.11570)

---

## Abstract Summary

This comprehensive survey covers 650+ South Asian languages across four families: Indo-Aryan, Dravidian, Iranian, and Tibeto-Burman. It examines the state of transformer-based NLP (BERT, T5, GPT variants) for these languages, analyzing data availability, model coverage, and downstream task performance. The survey finds that Indo-Aryan languages (Hindi, Bengali, Gujarati) are reasonably well-covered while Tibeto-Burman and Iranian-family languages in South Asia remain almost entirely absent from NLP research. It catalogs existing multilingual benchmarks, datasets, and models with specific coverage gaps.

---

## Key Contributions

- **Language census**: Documents 650+ South Asian languages with their NLP coverage status
- **Family-level analysis**: Coverage breakdown by Indo-Aryan, Dravidian, Tibeto-Burman, Iranian
- **Model survey**: Catalogs all existing multilingual models with South Asian language support
- **Gap identification**: Quantifies which language families are NLP-invisible
- **Resource catalog**: Links to datasets, corpora, and benchmarks for each covered language

## Relevance to indic-agents

Provides the **landscape view** of which Indic languages have sufficient NLP infrastructure to support the project's agent applications today versus which require data creation efforts first. Essential for prioritizing the project's language roadmap.
